# Evolution 游戏

## 功能特性
- 生物进化模拟系统
- 基因突变与自然选择机制
- 生态环境动态平衡
- 多线程模拟运算

## 游戏规则
1. 每个生物包含DNA序列
2. 环境因素影响生存概率
3. 繁殖周期自动产生新世代
4. 资源竞争决定种群规模

## 快速开始
### 环境要求
- JDK 17+
- Gradle 8.6+

### 构建运行
```shell
gradle build
gradle run
```

## 项目结构
```
src/main/java
├── core      # 游戏核心逻辑
├── model     # 数据模型定义
├── util      # 工具类
└── view      # 可视化模块
```

## 开发指南
- 使用`git flow`开发流程
- 遵循Google Java代码规范
- 单元测试覆盖率要求≥80%
